The x402 handshake

1 · GET
Agent requests a resource → server replies 402 Payment Required with amount, recipient, chain.
2 · Preview
Agent shows the price to the user and waits for confirmation. No spend yet.
3 · Pay
MUSD transfer on Mezo (explicit gas) → ~1s confirmation.
4 · Deliver
Retry with X-PAYMENT: {txHash} → server verifies on-chain → 200 + content.
